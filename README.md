# MongoDB Node.js Task

## Project Overview
This project demonstrates a simple Node.js backend application using **Express** and **MongoDB Atlas**. The application connects to a MongoDB database in the cloud and runs a server that confirms the connection.

---

## Features
- Connects to **MongoDB Atlas** using **Mongoose**
- Uses **Express** for server and routing
- Displays a success message when the server and database are connected
- Clean project structure with dependencies managed in `package.json`
- Ready for deployment on platforms like **Render**, **Railway**, or **Cyclic**

---

## Installation Instructions

1. Clone the repository:
git clone https://github.com/YOUR_USERNAME/mongo-task.git
2. Navigate to the project folder:
cd mongo-task
3. Install dependencies:
npm install
4. Start the server using Node.js:
node index.js
5. After running, the terminal will show:
Server running on port 3000
MongoDB Connected Successfully
6. Testing the Application
Open your browser and visit:
http://localhost:3000
You should see:
MongoDB Atlas Connected Successfully
